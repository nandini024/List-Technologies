FRONTEND

HTML

HTML (HyperText Markup Language) is the standard language for creating and designing webpages. It structures content using elements, represented by tags enclosed in angle brackets.

CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls the layout, design, and appearance of web pages, including elements like colors, fonts, spacing, and positioning. CSS allows developers to create visually appealing and responsive designs by separating content from design, enabling easier maintenance and scalability. It supports features like animations, transitions, and media queries for responsive designs

JAVASCRIPT

JavaScript is a versatile programming language primarily used for creating dynamic and interactive content on websites. It enables functionalities like form validation, animations, and real-time updates without reloading the page. JavaScript runs in web browsers and works alongside HTML and CSS to enhance user experiences.

DATABASE

MONGODB

MongoDB is a popular open-source, NoSQL database designed for storing and managing large volumes of unstructured or semi-structured data. It uses a document-oriented model, where data is stored in flexible JSON-like documents called BSON (Binary JSON).

BACKEND

JAVA

Java is a high-level, object-oriented programming language designed to be platform-independent through the "write once, run anywhere" (WORA) principle. Developed by Sun Microsystems (now owned by Oracle) in 1995, Java is widely used for building web, mobile, and desktop applications, as well as enterprise systems and backend services.

SPRINGBOOT

Spring Boot is a powerful, open-source framework built on top of the Spring Framework, designed to simplify the development of Java-based applications. It provides a streamlined approach to building production-ready applications with minimal configuration.

Frointend Frameworks

React.js: A library developed by Facebook for building user interfaces with reusable components.
Angular: A TypeScript-based framework developed by Google for creating dynamic, single-page applications (SPAs).
Vue.js: A lightweight framework for building user interfaces, offering simplicity and flexibility.
Svelte: A modern framework where components are compiled into efficient JavaScript at build time.
